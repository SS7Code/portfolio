Hello — I build computational tools for fracture mechanics and CAE workflows.

Virtual Element Method · Phase-field Fracture · Abaqus plugins

I’m completing my PhD on the Virtual Element Method for crack propagation. I develop research-grade implementations (Abaqus UELs, MATLAB), publish papers, and prototype CAE tools.

About

I research numerical methods for fracture mechanics, focusing on the Virtual Element Method (VEM) and its application to phase-field fracture simulations and cohesive interface models. I implement prototypes in Abaqus (UEL/UMAT) and MATLAB, and I am interested in turning research code into reproducible tools.

Selected projects

VEM for phase-field fracture

Implementation of lowest-order VEM with phase-field fracture; studies on mesh transition and hanging nodes.

Abaqus UEL: Virtual elements

Custom UEL for virtual elements supporting non-matching meshes and cohesive interfaces.

Prototype CAE plugin

Designing a plugin to streamline fracture simulations for reproducible workflows.
